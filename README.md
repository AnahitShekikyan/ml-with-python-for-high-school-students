# 🐍 Machine Learning with Python for High School Students

> A beginner-friendly course introducing Python programming and machine learning for students in grades 10–12. No prior experience required.

---

## 👋 Welcome

Have you ever wondered how Netflix recommends your next show, how Spotify builds playlists, or how phones recognize faces? These are examples of **machine learning**.

This course is designed for **high school students in grades 10–12** who want to learn Python and begin exploring machine learning.

The course starts with Python basics and gradually builds toward more advanced topics and real-world projects. By the end, students will have hands-on experience with important tools used in data science and machine learning.

---

## Who Is This Course For?

| Grade | What to expect |

| 10th grade | A strong starting point with Python fundamentals taught from the beginning |
| 11th grade | A chance to strengthen coding skills and begin learning machine learning concepts |
